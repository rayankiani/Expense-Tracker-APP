## ✨ Features

### 🎯 Core Features
- **📊 Dashboard** — At-a-glance view of your financial health with balance, income, and expenses
- **💳 Transaction Management** — Add, edit, and delete income/expense transactions with ease
- **🔍 Advanced Filtering** — Search by title, filter by type (income/expense), category, and date range
- **📈 Analytics Dashboard** — Visual insights with interactive charts

### 🎨 UI/UX Features
- **🌙 Dark/Light Theme** — Toggle between dark and light modes with smooth transitions
- **📱 Fully Responsive** — Works flawlessly on desktop, tablet, and mobile devices
- **✨ Smooth Animations** — Delightful micro-interactions and transitions throughout
- **🗓️ Real-time Date Display** — Always shows current date in the header

### 📊 Analytics Charts
- **🍩 Donut Chart** — Expense breakdown by category with animated segments
- **📊 Bar Chart** — Income vs Expenses comparison
- **📈 Line Chart** — Monthly overview for the last 6 months

### 💾 Data Management
- **🔒 Local Storage** — All data stored locally in your browser (no server required)
- **📤 Export Ready** — Data structure ready for future JSON export functionality

## 🖥️ Usage Guide

### Adding a Transaction
1. Click **"Add Entry"** in the sidebar
2. Select **Income** or **Expense**
3. Enter title, amount, date, and category
4. Click **"Add Transaction"**

### Editing a Transaction
1. Navigate to **Transactions** or view from Dashboard
2. Click the **✏️ edit icon** on any transaction
3. Modify the details
4. Click **"Save Changes"**

### Filtering Transactions
- Use the **search bar** to find by title or notes
- Click **filter pills** to filter by type (All/Income/Expense)
- Select **category pills** for category-specific filtering
- Use **date pickers** for date range filtering

### Viewing Analytics
- Click **"Analytics"** in the sidebar
- View expense breakdown by category
- Compare income vs expenses
- See monthly spending trends
